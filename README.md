# AQA Assembly (ASM)
Syntax Highlighting for the AQA Assembly Instruction set

## Features
Works on .asm, .as and .a files<br>
Keyword highlighting (ADD, SUB, CMP etc)<br>
Branches, registers, numbers and comments

(With Bearded Coffee Theme)<br>
